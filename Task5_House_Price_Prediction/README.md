#House Price Prediction with Linear Regression 

Built a predictive model to estimate house prices using Linear Regression.

# Dataset
Housing dataset with features like area, bedrooms, bathrooms, 
airconditioning, parking and more.

# What I did
- Explored and cleaned the dataset
- Encoded categorical columns (yes/no features)
- Split data into training and testing sets
- Trained Linear Regression model using Scikit-learn
- Evaluated model using R2 Score and MSE

#Result
- R2 Score: 0.649
- Model explains 64.9% of house price variation

#Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
